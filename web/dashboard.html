How this answers the 4 project problems

The interface is organized around the main educational needs of the project.

1
Understand mistakes It displays the visible concept, possible root cause, explanation and recommendation, instead of only a score.
2
Link errors to concepts The graph connects questions, AI concepts and prerequisites, making the diagnosis traceable.
3
Personalize support Each student has specific weaknesses and targeted recommendations.
4
Teacher dashboard Charts summarize frequent concepts, root causes and students needing attention.
