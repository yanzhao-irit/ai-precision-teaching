Teaching Dashboard

A simple teacher dashboard for the AI course diagnosis prototype. It uses the generated diagnosis CSV to explain student mistakes, identify possible prerequisite gaps, and prepare personalized support.

Link with the 4 project problems

1. Understanding mistakes
The dashboard does not only show wrong answers. It displays the visible concept, possible root cause, explanation and recommendation.
2. Linking errors to course concepts
Each failed question is linked to a concept, and each concept is linked to prerequisites in the knowledge graph.
3. Personalized support
Each student has a specific diagnosis and recommendations based on their own failed questions.
4. Teacher dashboard
The charts summarize frequent errors and root causes to help the teacher adapt the course.
