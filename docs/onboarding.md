## 协作约定 · Collaboration Norms

### 中文

1. **沟通渠道：**
   - 项目相关问题 → GitHub Issue
   - 紧急或私密事项 → 微信/邮件给 PI
   - 闲聊 → 项目微信群

2. **响应时间：**
   - PI 邮件：48 小时内回复
   - 同伴消息：尽量当天回
   - GitHub Issue：3 工作日内 review

3. **每周节奏：**
   - 周一上午：周会，汇报上周进度 + 本周计划
   - 周三下午：可选 standup (15 分钟，遇到的阻塞)
   - 周五下午：提交周报到 `docs/meetings/YYYY-MM-DD-weekly.md`

4. **遇到困难：**
   - 卡 2 小时 → 先 Google + 问 AI
   - 卡半天 → 在 GitHub Issue 写出来，找同伴 review
   - 卡一天 → 直接找 PI，不要硬扛

### English

1. **Communication channels:**
   - Project questions → GitHub Issue
   - Urgent or private → WeChat/email to PI
   - Casual → Project WeChat group

2. **Response times:**
   - PI emails: 48 hours
   - Peer messages: same day if possible
   - GitHub Issues: within 3 business days

3. **Weekly rhythm:**
   - Monday AM: Weekly meeting — last week's progress + this week's plan
   - Wednesday PM: Optional 15-min standup for blockers
   - Friday PM: Submit weekly report to `docs/meetings/YYYY-MM-DD-weekly.md`

4. **When stuck:**
   - Stuck 2 hours → Google + ask AI first
   - Stuck half a day → Open a GitHub Issue, ask peer to review
   - Stuck a full day → Go to PI directly, don't suffer alone
