## 邮件正文 · Email Body

```
主题：AI 赋能教学改革项目 · 研究生加入与协作方式说明

涂老师 / 陈老师 / 许老师 / 杨老师：

您好。

省级教改项目"AI 赋能课程学习错因诊断与靶向提质的关键技术与
实践研究"目前已进入实质推进阶段。近期有两位研究生加入项目，
作为当前阶段的主要劳动力。

为避免后续协作中出现不必要的打扰，提前向各位说明项目的协作
约定：

一、项目整体由我（赵妍）负责推进。所有任务安排、进度控制、
   对外接口都由我承担。

二、各位老师作为项目顾问，主要在关键节点提供专业指导，预计
   每月的项目相关咨询约 2-4 小时即可（具体根据各位老师在
   项目中的分工方向）。

三、研究生不会直接打扰各位。如有需要您们帮助的问题，会先
   汇总到我这里，由我整理后统一向各位请教或邮件咨询。

四、我们已经建立了 GitHub 项目仓库，所有阶段计划、文档、
   产出都在上面。如各位有兴趣随时关注进度，我可以将仓库
   访问权限发送给您们。

近期需要您们配合的事项：
- 涂老师：阶段 2（线性代数知识图谱）的知识点审核会议，
        预计 [日期] 至 [日期] 进行 1-2 次，每次 1.5 小时
- 许老师：知识图谱方法层面的咨询，邮件即可
- 陈老师：在系统架构选型上的把关咨询，邮件即可
- 杨老师：后续系统开发推进时再具体对接

如有疑问随时联系。感谢各位的支持。

赵妍
[日期]
```
